How to Update / Add New Project Link and Info (Latest):
----
1.Go to index html and add / update the relevant Project Cards.. (see <div class="projects-grid"> that's where the projects are)
Note: See "Empty Slide Test" as an example.

2. Go to loading_url_firebase.js.

3. Update the following below:
// 1a. Project IDs to load
// 1b. Other link IDs to update (if necessary)
// 2. Helper function to get project name from id (placeholder)
// 3. Fetch project titles and descriptions

4. Go to this website: https://firebase.google.com/docs/database

5. Click on "Go to Console"

6. Then, click on "Coding Portfolio Database"

7. Next, click on "Realtime Database" on the left sidebar.

8. Update the media and URL links as per the format.

9. You're done!

How to Save Files and Deploy to Netlify:
----
- Once you're happy with the changes, then save the files.
- Make a copy of the folder as a zip file.
- Meanwhile, upload the coding portfolio to Netlify (remember to login) manually.
- Wait for it to deploy and that's it!
